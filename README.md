# Campus Notifications Platform

A production-grade React application built with Material UI to aggregate, filter, and prioritize campus notifications dynamically.

## Core Features
* **Priority Inbox Algorithm:** Automatically anchors high-importance notifications (Placements, Results) to the top using a custom combined sorting weight formula.
* **Read/Unread Tracking:** Frontend tracking that visually distinguishes new alerts using visual anchor accents.
* **Advanced Controls:** Dynamic "n" capacity limit modifiers and categorical dropdown filters.
* **Production Observability:** Connected asynchronous logging pipeline tracking component states and network events.
